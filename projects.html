mr. collector
Design • Management • Development
mr.collector is an unofficial eshop built around Wordpress, WooCommerce and relative e-commerce plugins. Aside of the website, a business plan was created including SWOT Analysis, marketing strategies and business schemes to make a complete simulation of settling in the industry.
Speech to Morse Code
VIEW
Development • Design
Speech to Morse Code is a Matlab app using the Google Cloud Speech-to-Text API. The user can record themselves saying a single word or a more complicated sentence and get an output of how this recording sounds like in morse code with live audio configurations customizing a different result every time.
